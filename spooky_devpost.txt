G🌍 Project Go Go Green  ·  Go Clean  ·  Go Smart  ·  Go Earth AgentathonX 2026  ·  AI-Powered Waste Classification Agent for India 🇮🇳 62M Tonnes Waste Per Year 📸 3 Agents AI Calls Per Scan 🌍 11 of 17 SDGs Addressed 🪙 10 Impact Gifts Real Rewards The Problem India is one of the world's largest waste generators — 62 million tonnes a year. Yet most Indians don't know which bin to use. Not because they don't care, but because the system is confusing and there's no accessible tool to guide them in real time. The result: 80% of India's waste ends up in open dumps or rivers. A ₹1 lakh crore recycling economy sits untapped. Disease spreads. Groundwater is poisoned. Wildlife dies from plastic ingestion. "62 million tonnes of waste. Most Indians don't know which bin to use. Project Go fixes that with one photo." The Solution — Project Go Project Go is a full-stack AI waste classification agent. You photograph any piece of waste — Project Go does everything else. 📸 Snap a Photo Upload from gallery or use live camera with back-facing default, flip button, and scanning animation 🤖 AI Identifies It Gemini 2.0 Flash Vision analyzes the image — identifies item, classifies category, gives confidence score 🗑️ Bin Guidance Tells exactly which bin to use: Green (wet), Blue (dry), Red (hazardous), E-Waste Center, or Black (sanitary) 🌱 Environmental Impact Calculates carbon % saved, energy saved, water saved, wildlife protected, fun facts — all real data 📍 Nearest Centers Uses geolocation to find 3 nearest recycling centers, with Google Maps links 🪙 EcoCoins Earn coins per scan, unlock badges, redeem for real-world rewards like planting trees 💰 Kabadiwala Earnings Track real ₹ earned selling scrap — newspapers, plastic, iron, copper, old phones 🌍 SDG Education Learn how each scan connects to 11 UN Sustainable Development Goals AI Agent Architecture Project Go uses a 3-call agentic pipeline. Each call to Gemini 2.0 Flash is specialized: Agent Input Output analyze Base64 image Item name, category, confidence %, disposal method, eco tip, recyclability, decomposition days, India stat impact Item name + category Carbon %, energy saved, water saved, recycled into, trees equivalent, wildlife fact, fun fact centers Category + city from geolocation 3 nearest recycling centers: name, address, distance, type, tip, Maps link Security: The Gemini API key lives only in a Netlify serverless function (process.env.GEMINI_API_KEY). The frontend calls /api/ai — the key is never exposed to the browser.
