# ✦ Srishti Suman Gupta — Portfolio

A minimal, editorial portfolio built with **React + Vite**, featuring clean typography, smooth animations, and a magazine-inspired design aesthetic.

![React](https://img.shields.io/badge/React-19-61DAFB?style=flat-square&logo=react&logoColor=white)
![Vite](https://img.shields.io/badge/Vite-8-646CFF?style=flat-square&logo=vite&logoColor=white)
![CSS](https://img.shields.io/badge/Vanilla_CSS-Custom-1572B6?style=flat-square&logo=css3&logoColor=white)

## 🖼️ Design

- **Style** — Editorial, light-mode, monochromatic
- **Typography** — Instrument Serif (display) + Inter (body)
- **Palette** — Off-white background `#fafaf9` · Near-black text `#1a1a1f`
- **Layout** — Whitespace-heavy, magazine-inspired grid

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## 📂 Structure

```
src/
├── components/
│   ├── Navbar/          # Fixed top navigation
│   ├── Hero/            # Landing section with editorial title
│   └── SkillsMarquee/   # Infinite scrolling skills band
├── App.jsx              # Root component
├── index.css            # Design system & global styles
└── main.jsx             # Entry point
```
